## About This Repository

This is the source code of my personal portfolio — **[miftahulislamefaz.xyz](https://miftahulislamefaz.xyz)**

An immersive, motion-heavy, cinematic web experience built to represent who I am and what I build. No templates. No themes. Every pixel intentional.

---

## The Stack

```
React 18 + TypeScript + Vite
GSAP + ScrollTrigger + Lenis Smooth Scroll
Three.js (WebGL) + GLSL Shaders
Framer Motion
Tailwind CSS
Supabase
```

---

## What Makes It Different

- **WebGL 3D scene** — custom `.glb` model with particle nodes rendered via Three.js canvas
- **Physics-based service cards** — hanging from a steel rod with real inertia and swing mechanics
- **GooeyNav** — liquid SVG-filter morphing mobile menu with tactile feedback
- **RevealLoader** — typographic preloader with staggered wipe animations
- **FaviconAnimator** — browser tab reactive state indicator
- **Cinematic Hero** — multi-axis parallax, noise grids, scroll-velocity-reactive overlays
- **Torn paper transition** — custom visual separator between sections
- Full Lenis + GSAP ScrollTrigger integration across every scroll interaction
